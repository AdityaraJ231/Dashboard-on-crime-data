# Air Pollution Analysis Dashboard

## Overview

This project focuses on analyzing and visualizing air pollution data using Power BI. The dashboard provides meaningful insights into pollution levels, major pollutants, and trends across different regions. The aim of the project is to better understand air quality patterns and support data-driven environmental analysis.

## Features

* Interactive Power BI dashboard
* Air pollution trend analysis
* Region-wise pollution comparison
* Data visualization using charts and graphs
* KPI indicators for pollution levels
* Easy-to-understand insights for users

## Tools & Technologies Used

* Power BI
* Excel / CSV Dataset
* Data Cleaning & Transformation
* Data Visualization

## Key Insights

* Identified highly polluted regions based on AQI values
* Compared pollution levels across multiple locations
* Analyzed trends and patterns in air quality data
* Visualized pollutant contribution using charts and dashboards

## Project Objectives

* To analyze air pollution data effectively
* To create an interactive dashboard for visualization
* To understand pollution patterns and environmental impact
* To improve analytical and dashboard-building skills using Power BI
